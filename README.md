# Nepal Tourism Industry Analysis (1964–2023)

## Overview
An end-to-end business analytics project analysing Nepal's
international tourism industry using official government data
from the Ministry of Culture, Tourism & Civil Aviation.

## Tools Used
- Google Sheets — data cleaning and analysis
- Tableau Public — interactive dashboard
- Google Docs — business insights report
- Data Source: Nepal Tourism Statistics 2023
  (Ministry of Culture, Tourism & Civil Aviation)

## Live Dashboard
[View Interactive Dashboard on Tableau Public](https://public.tableau.com/app/profile/raj.hamal/viz/NepalTourismAnalysis/NepalTourismPerformanceDashboard19902023)

## Key Findings
1. Nepal tourism grew 10x from 93,000 (1975) to 1.2M (2019)
2. COVID-19 caused an 80.7% crash in 2020 — worst in history
3. Chinese arrivals only 40% recovered vs pre-COVID levels
4. Revenue recovery lags arrivals: USD 548M vs USD 771M peak
5. 2023 arrivals: 1,014,882 — on track to beat 2019 record by 2025

## Business Recommendations
1. Target China — biggest unrecovered market (60K vs 154K pre-COVID)
2. Raise spend per visitor from USD 41 to USD 60+ through premium products
3. Diversify beyond India (currently 31.5% of all arrivals)
4. Develop Gautam Buddha International Airport for new routes
